# Gallery

Where the [General](../general/introduction.md), [Widget](../widget/getting-started.md),
and [JS Library](../library/getting-started.md) sections explain *how CifVis works*,
the gallery shows *what you can build with it* — potential applications and how CifVis
plugs into a larger page. Every entry is a live demo plus the code to reproduce it, both
as widget markup and as library JavaScript.

| Entry | Shows |
|---|---|
| [Comparison views](./comparison-views.md) | Two synchronized viewers for comparing refinements or render styles. |
| [Difference density](./difference-density.md) | A progressive Fo−Fc map calculated in the browser from embedded reflection data. |
| [Publication-style 2D](./publication-2d.md) | A print-ready ORTEP-style figure with the `cutout-2d` renderer. |
| [Contour section](./contour-section.md) | Line-only density contours in a molecular plane. |
| [Custom theming](./custom-theming.md) | CSS custom properties plus element recolouring for a branded viewer. |
| [Disorder handling](./disorder.md) | Cycling disorder groups on a disordered structure. |
| [Labeled figures](./atom-labels-figure.md) | Maximum-coverage atom labels for an annotated figure. |
| [Embedding with other JS](./embedding.md) | A viewer coexisting with your own JavaScript — selection driving a chart. |
| [Extended solids](./extended-solid.md) | An ionic structure grown through symmetry with the unit cell displayed. |

All demos use the sample CIF files that ship with the
[playground](https://niolon.github.io/cifvis/).
