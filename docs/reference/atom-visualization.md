# Atom visualization settings

These options control the appearance of atoms in the 3D structure: geometry detail,
material properties, ADP ring geometry, and the cutaway hatching used by the
`cutout-3d` render style (the 2D hatch equivalents are `plot2DStripeCount` and
`plot2DStripeWidth` under [Rendering](./rendering.md)).
